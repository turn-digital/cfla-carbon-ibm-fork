# Back to

## Apraksts

Komponente "Back to" ir lietotāja saskarnes saite, kas ļauj lietotājiem
atgriezties iepriekšējā lapā vai ekrānā

## Anatomija

- Saite (bulta)
- Virsraksts, pēc noklusējuma “Back to”

## Dizaina un UX komentāri:

Lieto jebkur, kur nepieciešams pievienot saiti uz iepriekšējo skatu
