# Label Value

## Apraksts

Informācija par lauku ar rindu zemāk novietotu slotu, kurā ievietot papildus
informāciju teksta veidā

## Anatomija

- Label - nosaukums, pēc noklusējuma “Default label”
- Value - slotā ievietotā informācija, var būt teksts vai links

## Dizaina un UX komentāri

Tiek izmantots kontekstuālajā headerī, lai attēlotu statusu vēsturi.

## Getting started

### JS (via import)

```imoprt
<script type="module" src="https://demo.turn.lv/cfla_dist/turn-label-value.min.js"></script>
```

```html

  label="Propsed label!"
  value="Propsed Value!"></cds-label-value>
<cds-label-value

  value="Propsed Value!"></cds-label-value>

<cds-label-value label="Propsed label!">
  <p>Slot Value</p>
</cds-label-value>
```
