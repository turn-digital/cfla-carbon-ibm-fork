# Status history
## Apraksts
Komponents attēlo veidlapas pašreizējo statusu un statusu vēsturi, ja ir mainīts statuss.

## Anatomija
* currenStatusTitle  - šī brīža statuss
* currenStatusDate - šī brīza statusa iegūšanas datums
* Statusu vēsture -  statusu vēstures bloks
* statusHistoryItems - statusa vēstures ieraksts ar datumu, statusu, autoru, pamatojumu, statusa identifikācijas numuru

## Dizaina un UX komentāri
Izmanto veidlapas headerī, attēlo aktuālo statusu augšā un vēsturiskos statusus izvēršamā statusu vēstures blokā.
