# Related documents

## Getting started

Here's a quick example to get you started.

### Description
Related form with a status that is important to see before opening. The possibility to display a prompt for action, for example, review, compare, fill out.


### JS (via import)

```javascript
<script type="module" src="https://demo.turn.lv/cfla_dist/turn-related-documenents.min.js"></script>
```



### HTML

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <script
      type="module"
      src="https://demo.turn.lv/cfla_dist/turn-related-documenents.min.js"></script>
    <link
      rel="stylesheet"
      href="https://1.www.s81c.com/common/carbon-for-ibm-dotcom/tag/latest/plex.css" />
    <title>Related documents</title>
  </head>

  <body>
    <cds-related-documents title="Saistītie dokumenti">
      <cds-related-documents-item
        linkTitle="Saistītais PI"
        linkUrl="https://google.com"
        target="_blank"
        linkIcon=${linkIconToolBox}>
      </cds-related-documents-item>
      <cds-related-documents-item
        linkTitle="Pārbaudes lapa"
        linkUrl="https://google.com"
        target="_blank"
        linkIcon=${linkIconCert}>
      </cds-related-documents-item>
      <cds-related-documents-item
        linkTitle="Saistītā līguma konsolidētā versija"
        linkUrl="https://google.com"
        target="_blank"
        linkIcon=${linkIconLicense}>
      </cds-related-documents-item>
      <cds-related-documents-item
        linkTitle="Saistītā PI grozījumu pamatojums"
        linkUrl="https://google.com"
        target="_blank">
      </cds-related-documents-item>
    </cds-related-documents>
  </body>
</html>
```
