# Related documents

## Getting started

Here's a quick example to get you started.

### Description

Related documents which are relevant to this form and which may need to be
consulted at the same time as the form.

### Anatomy

Heading 'Saistītie dokumenti' Document icon Link to document Launch\* icon opens
in a new window

### Design and UX comments

Document always opens in a new window The document icon and the document name
are linked:

- Tool-box\* icon - Related PI
- Certificate-check\* - all other statuses in between
- License* - Consolidated version of the linked contract Other document types do
  not have hanging icons *Icons are from the IBM Design Language Library
