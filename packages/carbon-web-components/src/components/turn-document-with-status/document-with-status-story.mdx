# Document with status

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
<script type="module" src="https://demo.turn.lv/cfla_dist/turn-document-with-status.min.js"></script>
```



### HTML

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <script
      type="module"
      src="https://demo.turn.lv/cfla_dist/turn-document-with-status.min.js"></script>
    <link
      rel="stylesheet"
      href="https://1.www.s81c.com/common/carbon-for-ibm-dotcom/tag/latest/plex.css" />
    <title>Document with status</title>
  </head>

  <body>
    <cds-document-with-status
      title="Projekta satura izmaiņu salīdzināšana"
      status="success"
      statusText="Izmaiņas ir veiktas"
      linkUrl="https://google.com"
      linkText="Apskatīt">
    </cds-document-with-status>
  </body>
</html>
```
