# Empty state
## Apraksts
Komponents tiek lietots, lai attēlotu situāciju, kurā vēl nav pievienoti saistītie dokumenti

## Anatomija
* Virsraksts “Dokumenti” (Slot=title)
* Poga “Pievienot”
* Tabula ar kolonnām “Nosaukums”, “Pievienošanas datums”,”Pievienotājs”
* Teksts tabulā “Šobrīd nav veikti līguma grozījumi” (empty state disabled stāvoklī parādās pievienotie dokumenti ar nosaukumu, pievienošanas datumu un pievienotāju)
* Saite “Pievienot” (empty state disabled stāvoklī netiek rādīts)

## Dizaina un UX komentāri
