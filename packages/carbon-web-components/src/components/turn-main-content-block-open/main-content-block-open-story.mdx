# Main content block open
## Apraksts
Komponente - ietvars. Tiek lietots atvērtas veidlapas skatā.

## Anatomija
* Tumšāks fona laukums header bloka daļā
* Slot - Header block
* Gaiši pelēks fona laukums satura bloka daļā
* Slot - Content block

## Dizaina un UX komentāri
Tiek izmantots veidlapas gadījumā kā pamats tālākam saturam, vizuāli atdalot header bloku no satura bloka ar atšķirīgas krāsas ietvara laukumiem. 

## Getting started
### JS (via import)

```imoprt
<script type="module" src="https://demo.turn.lv/cfla_dist/turn-main-content-block-open.min.js"></script>
```

### HTML

```html
<cds-main-content-block-open>
  <span slot="header">Header Block</span>
  <span slot="content">Content Block</span>
</cds-main-content-block-open>
```
