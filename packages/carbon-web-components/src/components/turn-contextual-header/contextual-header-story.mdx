# Contextual header

## Apraksts

Kontekstuāls hederis - attēlo atrašanās vietu sistēmā, piemēram, projekta
nosaukumu, kurš šobrīd redzams. Atverot dropdown daļu, iespējams iegūt vairāk
informācijas par konkrēto vienumu.

## Anatomija

- Virsraksts
- Chevron
- Dropdown daļa ar slotu

## Dizaina un UX komentāri

Kontekstuālais headeris, kas tiek lietots atlasē, projektā, projekta iesniegumā.
