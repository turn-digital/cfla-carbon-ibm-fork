# Status change panel

## Apraksts

Pogu bloks, ar kuru sistēmā tiek mainīts vērtēšanas statuss.

## Anatomija

- Poga “Saglabāt”
- Dropdown button “Mainīt statusu”

## Dizaina un UX komentāri

Komponents tiek izmantots, lai mainītu vērtēšanas statusu veidlapai.
