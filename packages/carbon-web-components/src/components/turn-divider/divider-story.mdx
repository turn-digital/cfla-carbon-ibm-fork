# Divider

## Apraksts
Divider - sadalītāja komponente atdala saturu vai elementus lapā, uzlabojot dizaina pārskatāmību un organizāciju.

## Anatomija
Atstarpe ar noteiktu izmēru, atbilstoši [carbon design system spacing vadlīnijām.](https://carbondesignsystem.com/elements/spacing/overview/)  
Sadalītāja īpašības (properties):
* Size, pēc noklusējuma ar vērtību 5. Var tikt iestatīta vērtība no 1 līdz 13.

## Dizaina un UX komentāri

Tiek lietota jebkur, kur pēc dizaina nepieciešama atstarpe.

## Getting started



### JS (via import)

```javascript
<script type="module" src="https://demo.turn.lv/cfla_dist/turn-divider.min.js"></script>
```



### HTML

```html

<html lang="en">
  <head>
    <meta charset="UTF-8" />

      src="https://demo.turn.lv/cfla_dist/turn-divider.min.js"></script>

      rel="stylesheet"
```
