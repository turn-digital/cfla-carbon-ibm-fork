# Label underline

## Getting started

### JS (via import)

```imoprt
<script type="module" src="https://demo.turn.lv/cfla_dist/turn-label-underline.min.js"></script>
```

### HTML

```JS
 const notifications = [
    { type: 'red', count: 2 },
    { type: 'green', count: 1 },
  ];
```

```html
<cds-label-underline
  label="Apraksts"
  .notifications="${notifications}">
</cds-label-underline>
```
