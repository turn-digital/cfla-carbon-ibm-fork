# Related persons
## Apraksts
Saisītās personas. Atkarībā no skata, kurā tiek lietots, var būt viena saistīitā persona vai vairākas. Vairākām personām tiek norādīti lomu saisinājumumi.

## Anatomija
* Ikona
* Lomas saīsinājums - tiek definēts katrai veidlapai individuāli
* Vārds uzvārds

## Dizaina un UX komentāri
Komponents tiek lietots:
* līgumos, norādot līguma veidotāju,
* projekta iesnieguma vērtēšanas veidlapā, norādot vienu vai vairākas saistītās personas.

## Getting started

### JS (via import)

```imoprt
<script type="module" src="https://demo.turn.lv/cfla_dist/turn-related-persons.min.js"></script>
```

```JS
const statusHistoryPersons = [

    { shortRole: 'AV', personName: 'Vards Uzvards', iconType: '2' },
    { shortRole: 'BN', personName: 'Vards Uzvards', iconType: '3' },
  ];
```

### HTML

```html

  .statusHistoryPersons="${statusHistoryPersons}"></cds-related-persons>
```
