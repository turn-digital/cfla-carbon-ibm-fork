# Project card
## Apraksts
Projekta kartīte. Tajā tiek attēlota svarīgākā informācija par projektu. 

## Anatomija
* CardTitle - Projekta nosaukums
* projectCostInfo - Projekta kopējās izmaksas
* ProjectCostTitle  - nosaukums projekta kopējām izmaksām, pēc noklusējuma “Projekta kopējās izmaksas EUR”
* ProjectDateInfo - projekta izpildes termiņš no kāda datuma līdz kādam datumam
* ProjectDateTitle  - nosaukums projekta izpildes termiņam, pēc noklusējuma “Projekta īstenošanas termiņš”
* ProjectId - projekta identifikācijas numurs (augšējā kreisajā stūrī)
* supervisorAuthority - atbalsta sniedzēja nosaukums (augšējā labajā stūrī), pēc noklusējuma “CFLA”
* tagTitle - statuss, piemēram, “Apstiprināts ar nosacījumu”

## Dizaina un UX komentāri
Tiek lietota vērtējuma pārskatīšanas izveidošanas skatā.
