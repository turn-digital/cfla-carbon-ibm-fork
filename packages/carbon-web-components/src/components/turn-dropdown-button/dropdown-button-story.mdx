# Dropdown Button
## Apraksts
Dropdown button ir komponents, kas ļauj lietotājiem izvēlēties vienu opciju no nolaižamajā izvēlnē attēloto opciju saraksta. Noklikšķinot uz pogas, tā izplešas uz leju, atklājot atlasāmo elementu sarakstu. Kā pogas teksts parādīta noklusējuma tekstu, kas aicina lietotājus izdarīt izvēli.  

## Anatomija
* Dropdown handler - poga ar noklusējuma tekstu.
* Accordion items - atlasāmo elementu saraksts, kas atklājas, noklikšņinot uz pogas.
* Chevron - bultiņa, kas pēc noklusējuma vērsta uz leju, bet atvērta elementu saraksta gadījumā  - uz augšu.

## Dizaina un UX komentāri
Dropdown button lieto gadījumā, ja ir iespējamas vairākas darbības, bet nav skaidra primārā. Statusu maiņas gadījumā - ja iespējami vairāki statusi, no kuriem izvēlēties, nevis tikai viens iespējamais nākmais statuss.

## Getting started

### JS (via import)

```imoprt
<script type="module" src="https://demo.turn.lv/cfla_dist/turn-dropdown-button.min.js"></script>
```

### HTML

```JS
 const itemsArray = [
    { text: 'Status 1', value: 'Status 1' },
    { text: 'Status 2', value: 'Status 2' },
    { text: 'Status 3', value: 'Status 3' },
  ];
  const eventHandler = (event) => {
    console.log('itemSelected', event.detail.item.__value);
  };
```

```html
<cds-dropdown-button
  .accordionItems="${itemsArray}"
  .dropdownHandler="${eventHandler}">
</cds-dropdown-button>
```
